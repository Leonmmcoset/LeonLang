# 欢迎使用 LeonBasic 编程语言

LeonBasic 是一种简单易用的解释型编程语言，专为初学者和快速原型开发设计。它提供了简洁明了的语法，同时支持基本的编程功能，如变量、函数、控制流、文件操作和网络功能等。

## 主要特性

* **简洁的语法** - 易于学习和使用
* **内置库支持** - 包含基础操作和网络功能
* **灵活的变量系统** - 支持字符串、整数和浮点数
* **完善的控制流** - 包括条件语句和循环结构
* **文件操作** - 支持文件的读写和追加
* **网络功能** - 支持 HTTP 请求和文件下载
* **错误处理** - 提供异常捕获机制
